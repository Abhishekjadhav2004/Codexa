# CodeForge Frontend

This is the frontend application for CodeForge - a MERN-based coding platform with real-time collaborative features.

## Features
- Modern React application with Vite
- Real-time collaborative coding with CodeForge Studio
- Dark/Light theme support
- Responsive design with TailwindCSS
- State management with Redux Toolkit

## Tech Stack
- React 19
- Vite
- TailwindCSS
- Redux Toolkit
- Socket.IO Client
- CodeMirror Editor
- Framer Motion

## Getting Started

```bash
npm install
npm run dev
```

## Available Scripts
- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
- `npm run lint` - Run ESLint
